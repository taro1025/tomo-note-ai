# tomo-note-ai

TomoNote AI の App Store 提出用マーケティングページとプライバシーポリシーです。

## Pages

- `index.html`: マーケティングページ
- `privacy.html`: プライバシーポリシー
- `styles.css`: 共通スタイル
- `assets/app-icon.png`: App Store / Web 表示用アイコン

## GitHub Pages

GitHub リポジトリの `Settings` → `Pages` で、`Deploy from a branch` を選び、`main` ブランチの `/root` を公開元に設定します。

公開後は以下のURLを App Store Connect に設定できます。

```text
Marketing URL: https://taro1025.github.io/tomo-note-ai/
Privacy Policy URL: https://taro1025.github.io/tomo-note-ai/privacy.html
```
